# 部落格功能設計 — 「Aria 老師碎碎念」

> 狀態:設計已與使用者確認(2026-06-09),待 spec 審閱後進入實作計畫。
> 網站:https://tingyumusic.com/(純靜態單頁,GitHub Pages,無建置流程)

## 1. 目標與對象

為「聽聽音樂 / 郭庭羽 Aria Kuo」教學網站新增部落格,以**長尾關鍵字內容**強化 SEO(對應原 SEO 執行方案的 Phase 4-B),同時提供潛在客戶有用資訊、建立專業信任、把讀者導回預約。

- **主要對象**:① 學生家長 ② 成人音樂學習者
- **語氣**:第一人稱、溫暖、口語、像老師在跟你聊天(呼應「碎碎念」品牌名);不要寫成生硬的教科書條列。
- **每篇**:約 1000 字,圖文並茂(2~3 張插畫)。

## 2. 架構與檔案結構(沿用現有設計,零新依賴)

```
/blog/index.html            → 部落格首頁(文章列表卡片)
/blog/<slug>.html           → 每篇文章獨立 HTML(獨立 URL / title / SEO)
/assets/blog/<slug>/        → 該篇插畫(cover.png + inline-*.png,1024×1024)
/scripts/gen-illustration.sh → Gemini 插畫生成腳本(專案內,方便日後補圖)
```

- 設計語言完全沿用現有 `index.html` 的 CSS 變數與字體(米色 `--ivory`、金 `--gold`、Noto Serif/Sans TC、Cormorant),不引入框架或 build。
- 文章頁的共用樣式抽到各 HTML 內的 `<style>`(與現站一致的內嵌做法);可接受少量重複,維持「無建置」原則。

## 3. 導覽與動線

- 主站 `index.html` 導覽列新增連結:**Aria 老師碎碎念** → `/blog/`
- 部落格首頁與文章頁皆保留回主站的導覽(logo 連回 `/`)。
- 每篇文章結尾放 **CTA 區塊**:「想開始學長笛/鋼琴?」+ Line 預約按鈕,導回主站聯絡。
- 文章頁彼此可互連(文末「延伸閱讀」放 2~3 篇相關文章連結)。

## 4. 文章頁模板結構

固定骨架,讓 10 篇一致:

1. 頂部導覽(同主站精簡版)
2. 文章標頭:麵包屑(首頁 / Aria 老師碎碎念 / 文章標題)、`<h1>` 標題、發布日期、一句導言
3. **封面插畫**(cover.png)
4. 前言段落
5. 2~3 個 `<h2>` 小標題段落,段落間穿插 inline 插畫
6. 「重點整理」小結(條列 3~5 點)
7. CTA 區塊(預約/Line)
8. 延伸閱讀(相關文章連結)
9. footer(同主站)

## 5. 視覺與插畫

- **插畫風格**:暖色扁平插畫(flat illustration, pastel colors, soft shadows, rounded shapes, no text, no letters),與本次測試圖一致。
- **每篇 2~3 張**:1 張封面 + 1~2 張內頁示意。
- **產圖管道(已實測可行)**:
  - `curl` 呼叫 Gemini `gemini-2.5-flash-image:generateContent`,輸出 1024×1024 PNG。
  - 依賴:`GEMINI_API_KEY`(✅ 已設定)、`jq`(✅)、`python3`(抽 base64)。
  - 寫成 `scripts/gen-illustration.sh "<prompt>" <輸出路徑>`,風格基底自動附加,確保跨篇一致。
- 插畫**不含文字**(Gemini 中文渲染不穩);所有文字訊息由文章本文承載。

## 6. SEO 規格(每篇文章)

- 獨立 `<title>`(含關鍵字 + 客群)、`<meta name="description">`
- `<link rel="canonical">` 指向該篇完整網址
- Open Graph(og:title/description/image=該篇封面/url/type=article)
- **Article JSON-LD**:`@type: Article`、headline、datePublished、image、`author`(Person 郭庭羽 Aria Kuo,參照主站 `#teacher`)、`publisher`(聽聽音樂,參照 `#business`)、mainEntityOfPage。
- `/blog/index.html` 與每篇文章 URL 皆加入 `sitemap.xml`(`changefreq` 文章用 `yearly`/首頁 `monthly`)。
- 部落格首頁從主站導覽列連入(內部連結)。

## 10. 風險與備註

- **Gemini 插畫風格一致性**:靠固定 prompt 風格基底控制;若某張偏離,重產即可(API 成本低)。
- **無文字插畫**:若某概念一定要圖上文字(例如五線譜示意),改用簡單標註或交由本文說明,不在插畫內嵌中文。
- **內容正確性**:文章涉及學習建議(年齡、練習方法等),以通用、穩健、不誇大的說法撰寫;具體價格/檢定細節保持彈性、引導預約諮詢。
